# Projekt_Architektury_nowoczesnych_system-w_informatycznych

- Celem projektu AirQ jest zaprojektowanie i implementacja webowego systemu eksperckiego do monitorowania jakości powietrza na terenie całej Polski. System ma za zadanie gromadzenie danych pomiarowych z zewnętrznej platformy OpenAQ, przetwarzać je przy użyciu zdefiniowanych reguł eksperckich oraz dostarczać użytkownikom końcowym rzetelne informacje, alerty i spersonalizowane rekomendacje dotyczące stanu powietrza w ich okolicy.
